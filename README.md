# Traditional RAG Chatbot

This project implements a Traditional Retrieval-Augmented Generation (RAG) chatbot using:

- Sentence Transformers
- FAISS Vector Database
- Chunking
- Embeddings
- Google FLAN-T5
- Gradio Chat UI

Workflow:

Text → Chunking → Embeddings → FAISS → Retrieval → Prompt → LLM Response

Built in Google Colab.
